Installation

  • 1. Getting the source

    You can access this model by installing the sofa-core package. Installing this package is as easy as running one of the following commands:

    • Via npm:

      npm install sofa-core

    • Via Bower:

      bower install sofa-core

    • Via Git:

      git clone http://github.com/sofa/sofa-core

  • 2. Embedding the source

    After that, embed the distribution file in your HTML document.

    <script src="dist/sofa.core.js"></script>

    Don't forget that each package defines its own package dependencies. Make sure to embed their files too if needed.

  • 3. Use it

    After that, you can inject this service by using Angular's dependency injection system:

    angular.module('yourApp').controller('FooController', ['sofa.models.Product', function (sofa.models.Product) {

}]);
