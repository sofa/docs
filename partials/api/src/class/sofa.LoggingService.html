Installation

  • 1. Getting the source

    You can access this class by installing the sofa-logging-service package. Installing this package is as easy as running one of the following commands:

    • Via npm:

      npm install sofa-logging-service

    • Via Bower:

      bower install sofa-logging-service

    • Via Git:

      git clone http://github.com/sofa/sofa-logging-service

  • 2. Embedding the source

    After that, embed the distribution file in your HTML document.

    <script src="dist/sofa.loggingService.js"></script>

    Don't forget that each package defines its own package dependencies. Make sure to embed their files too if needed.

  • 3. Use it

    After that, create an instance of this service by using the `new` operator:

    var loggingService = new sofa.LoggingService(configService);

Description

This service abstracts the concrete console interface away. It provides the same methods for logging like .log(), .info() etc..

Use this service to log within your application.
