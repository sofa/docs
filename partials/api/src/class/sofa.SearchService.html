Installation

  • 1. Getting the source

    You can access this class by installing the sofa-search-service package. Installing this package is as easy as running one of the following commands:

    • Via npm:

      npm install sofa-search-service

    • Via Bower:

      bower install sofa-search-service

    • Via Git:

      git clone http://github.com/sofa/sofa-search-service

  • 2. Embedding the source

    After that, embed the distribution file in your HTML document.

    <script src="dist/sofa.searchService.js"></script>

    Don't forget that each package defines its own package dependencies. Make sure to embed their files too if needed.

  • 3. Use it

    After that, create an instance of this service by using the `new` operator:

    var searchService = new sofa.SearchService(configService, httpService, qService);

Description

Search service which let's you query against the CouchCommerce API to search for products.

Methods

  • search(searchStr, grouping)

    Defined in sofa-search-service/src/sofa.searchService.js:25

    Searches for searchStr and groups the results if grouping is truthy. This search is promise based to let you have flow control. Therefore it returns a promise that gets resolved with the search results.

    Parameters

    Param Type Details
    searchStr string

    A search string.
    grouping boolean

    Whether to group the results or not.

    Returns

    Type Details
    Promise

    A promise with the search results.
