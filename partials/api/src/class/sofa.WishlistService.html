Installation

  • 1. Getting the source

    You can access this class by installing the sofa-wishlist-service package. Installing this package is as easy as running one of the following commands:

    • Via npm:

      npm install sofa-wishlist-service

    • Via Bower:

      bower install sofa-wishlist-service

    • Via Git:

      git clone http://github.com/sofa/sofa-wishlist-service

  • 2. Embedding the source

    After that, embed the distribution file in your HTML document.

    <script src="dist/sofa.WishlistService.js"></script>

    Don't forget that each package defines its own package dependencies. Make sure to embed their files too if needed.

  • 3. Use it

    After that, create an instance of this service by using the `new` operator:

    var wishlistService = new sofa.WishlistService();

Description

sofa.WishlistService is the interface to interact with a wishlist. It provides methods to add and remove items. It also takes care of writing updates to an available storage service.
