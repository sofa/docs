Installation

  • 1. Getting the source

    You can access this class by installing the sofa-url-parser-service package. Installing this package is as easy as running one of the following commands:

    • Via npm:

      npm install sofa-url-parser-service

    • Via Bower:

      bower install sofa-url-parser-service

    • Via Git:

      git clone http://github.com/sofa/sofa-url-parser-service

  • 2. Embedding the source

    After that, embed the distribution file in your HTML document.

    <script src="dist/sofa.urlParserService.js"></script>

    Don't forget that each package defines its own package dependencies. Make sure to embed their files too if needed.

  • 3. Use it

    After that, create an instance of this service by using the `new` operator:

    var urlParserService = new sofa.UrlParserService(locationService);

Description

This service provides a clean interface when it comes to accessing url ids for categories and products.
