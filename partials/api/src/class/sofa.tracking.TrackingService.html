Installation

  • 1. Getting the source

    You can access this class by installing the sofa-tracking package. Installing this package is as easy as running one of the following commands:

    • Via npm:

      npm install sofa-tracking

    • Via Bower:

      bower install sofa-tracking

    • Via Git:

      git clone http://github.com/sofa/sofa-tracking

  • 2. Embedding the source

    After that, embed the distribution file in your HTML document.

    <script src="dist/sofa.tracking.js"></script>

    Don't forget that each package defines its own package dependencies. Make sure to embed their files too if needed.

  • 3. Use it

    After that, create an instance of this service by using the `new` operator:

    var trackingService = new sofa.tracking.TrackingService(window, httpService, configService);

Description

Abstraction layer to communicate with concrete tracker services like Google Analytics.
