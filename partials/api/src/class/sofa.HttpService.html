Installation

  • 1. Getting the source

    You can access this class by installing the sofa-http-service package. Installing this package is as easy as running one of the following commands:

    • Via npm:

      npm install sofa-http-service

    • Via Bower:

      bower install sofa-http-service

    • Via Git:

      git clone http://github.com/sofa/sofa-http-service

  • 2. Embedding the source

    After that, embed the distribution file in your HTML document.

    <script src="dist/sofa.HttpService"></script>

    Don't forget that each package defines its own package dependencies. Make sure to embed their files too if needed.

  • 3. Use it

    After that, create an instance of this service by using the `new` operator:

    var httpService = new sofa.HttpService(qService);
