Installation

  • 1. Getting the source

    You can access this class by installing the sofa-device-service package. Installing this package is as easy as running one of the following commands:

    • Via npm:

      npm install sofa-device-service

    • Via Bower:

      bower install sofa-device-service

    • Via Git:

      git clone http://github.com/sofa/sofa-device-service

  • 2. Embedding the source

    After that, embed the distribution file in your HTML document.

    <script src="dist/sofa.deviceService.js"></script>

    Don't forget that each package defines its own package dependencies. Make sure to embed their files too if needed.

  • 3. Use it

    After that, create an instance of this service by using the `new` operator:

    var deviceService = new sofa.DeviceService();

Description

This is a helper service that gives you methods to check for certain contexts on touch devices etc. It determines the state for the usage of flexbox as well as things like overflow:scroll support.
